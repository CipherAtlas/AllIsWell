## Icons You Need to Add

### 1. PWA Icons (For "Add to Home Screen")

You need **2 PNG files** for the PWA manifest:

- `icon-192.png` (192x192 pixels)
- `icon-512.png` (512x512 pixels)

**Where to put them:** In the `public/` folder

**How to create them:**
1. If you have the original PNG/design file:
   - Resize to 192x192 and save as `icon-192.png`
   - Resize to 512x512 and save as `icon-512.png`
   
2. If you only have the .ico file:
   - Use an online converter: https://convertio.co/ico-png/
   - Or use ImageMagick: `magick AllisWell.ico -resize 192x192 icon-192.png`
   - Or use any image editor (GIMP, Photoshop, etc.)

### 2. Android App Icons (Optional - for better APK)

If you want custom icons for the Android APK, you'll need multiple sizes. But the default Capacitor icons work fine too!

**Required sizes for Android:**
- 48x48 (mdpi)
- 72x72 (hdpi)
- 96x96 (xhdpi)
- 144x144 (xxhdpi)
- 192x192 (xxxhdpi)

**Where:** `android/app/src/main/res/mipmap-*/ic_launcher.png`

**Note:** Android Studio can generate these automatically from a single 512x512 icon.

## Quick Setup

### Option 1: Use Your .ico File

If you have the original PNG/design:
1. Create `icon-192.png` (192x192)
2. Create `icon-512.png` (512x512)
3. Place both in `public/` folder

### Option 2: Convert from .ico

1. Convert `AllisWell.ico` to PNG using:
   - Online: https://convertio.co/ico-png/
   - Or any image editor
2. Resize to 192x192 → save as `icon-192.png`
3. Resize to 512x512 → save as `icon-512.png`
4. Place both in `public/` folder

### Option 3: Use a Placeholder (Temporary)

If you don't have icons ready, you can use simple colored squares temporarily:

```bash
# Create simple placeholder icons (requires ImageMagick)
magick -size 192x192 xc:#667eea icon-192.png
magick -size 512x512 xc:#667eea icon-512.png
```

Or use an online tool like: https://www.favicon-generator.org/
